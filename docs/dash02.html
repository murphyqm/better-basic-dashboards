Embedding html graphics

The Bokeh library allows you to save figures as html files, with limited in-browser interactivity. These figures can be saved as html output, which can then be included as a figure in your Quarto document or dashboard as an iframe component. While this is not ideal for fluid layout (it can be challenging to resize the iframe and embedded content correctly in order to fit these nicely), it does allow for inclusion of alternative text within the iframe tag.
